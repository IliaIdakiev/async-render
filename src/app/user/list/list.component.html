Single async-render (directive) with multiple resolvers. We also have a loader component that is used when fetching any of the data that needs to be resolved within the async-render. In this example we are loading all the posts for the selected user from the table. We also have mutiple buttons using the ViewChild selectors to imperativelly trigger a reload. We are also skipping the user posts resolver when there is no user selected.
